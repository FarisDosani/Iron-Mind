# Iron-Mind 

A front-end web application dedicated to fitness and exercise instruction, featuring a library of workout demonstration videos organized by muscle group and a built-in BMI calculator.

## 🌟 Features
* **Targeted Workouts:** Dedicated pages for Abs, Arms, Chest, Legs, and Shoulders.
* **Video Library:** Dozens of embedded MP4 demonstrations for exercises ranging from compound lifts to isolation movements.
* **Health Tracking:** An integrated BMI calculator to help users track their fitness metrics.

## 🛠️ Tech Stack
* **Markup:** HTML5
